 // A SIMPLE VOXEL ENGINE MADE IN C++ WITH OPENGL

FEATURES: 

	- INFINITE WORLD RENDERING
	- BLOCK PLACE AND DESTROY
	- SAVED GAME DATA


KEY BINDINGS:

-> MOVEMENT - W/A/S/D

-> UP : SPACE

-> DOWN : LEFT-SHIFT

-> PLACE BLOCK : LEFT-MOUSE BUTTON

-> DESTROY BLOCK : RIGHT-MOUSE BUTTON

-> BLOCKS : NUMS-1 - DIRT BLOCK
 	    NUMS-2 - STONE BLOCK
	    NUMS-3 - SAND BLOCK
